This is control software, written in Node.js, for the SumoBot Jr. It allows you to control the robot with your computer's keyboard.

In order to use this software, you will need to install Node.js on your computer. Node.js is a popular real time server-side scripting language, based on Google's JavaScript engine. To download your copy, head over to nodejs.org and press the big green install button.

To run this software after installing Node.js, simply open your computer's terminal application, 'cd' to this folder, and type in "node index.js".
